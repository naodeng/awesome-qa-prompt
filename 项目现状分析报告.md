## 1. 项目概览

### 1.1 基本信息

- **项目名称**: Awesome QA Prompt Collection
- **项目定位**: 专业的 QA（质量保证）AI 提示词集合
- **目标用户**: 测试工程师、QA 团队、软件质量保证专业人员
- **核心价值**: 提供结构化、专业的提示词模板，帮助测试工程师提升测试效率、确保测试覆盖、标准化流程、知识传承

### 1.2 技术栈

- **文档框架**: VitePress 1.0.0
- **运行环境**: Node.js 20+
- **构建工具**: npm
- **部署方式**: GitHub Actions 自动部署到 GitHub Pages
- **分析工具**: Umami Analytics
- **版本控制**: Git + GitHub

### 1.3 项目规模

- **测试类型模块**: 15 个
- **工作流程文档**: 3 个
- **提示词文件**: 60+ 个（完整版 + 精简版 + 中英文）
- **文档页面**: 100+ 个
- **代码行数**: 约 10,000+ 行（包含配置和文档）
