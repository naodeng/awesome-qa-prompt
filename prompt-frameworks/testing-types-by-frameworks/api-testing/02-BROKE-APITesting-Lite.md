# API Testing Prompt - BROKE Framework (Lite)

## Framework Context
The BROKE framework focuses on Background, Role, Objectives, Key Results, and Evolve.

## Prompt

```markdown
**Background:** Need to verify API quality for a new service release.
**Role:** Senior API Testing Expert.
**Objectives:** Create a concise API testing plan covering Functionality and Security.
**Key Result:** A Markdown table of test cases with Request/Response examples and specific validation steps.
**Evolve:** Refine test cases based on specific API constraints (e.g., Rate Limiting).
```
