# API Testing Prompt - CHAT Framework (Lite)

## Framework Context
The CHAT framework focuses on Character, History, Ambition, and Task.

## Prompt

```markdown
**Character:** Senior API Tester.
**History:** Need to test a newly developed API endpoint.
**Ambition:** Ensure the API works as expected and handles errors gracefully.
**Task:** Write a set of API test cases including Positive, Negative, and Boundary scenarios. Output in Markdown format with JSON body examples.
```
