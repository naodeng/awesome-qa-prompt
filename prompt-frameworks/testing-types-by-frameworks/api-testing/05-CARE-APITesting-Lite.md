# API Testing Prompt - CARE Framework (Lite)

## Framework Context
The CARE framework focuses on Context, Action, Result, and Example.

## Prompt

```markdown
**Context:** Need to verify a new API endpoint.
**Action:** Write test cases covering valid inputs, invalid inputs, and boundary conditions.
**Result:** A list of test scenarios in Markdown table format.
**Example:** Input: `age=-1`, Expected: `400 Bad Request`.
```
