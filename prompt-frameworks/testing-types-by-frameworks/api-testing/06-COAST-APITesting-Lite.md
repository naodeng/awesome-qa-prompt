# API Testing Prompt - COAST Framework (Lite)

## Framework Context
The COAST framework focuses on Context, Objective, Action, Support, and Technology.

## Prompt

```markdown
**Context:** Need to test a REST API for a user management system.
**Objective:** Verify that Create, Read, Update, and Delete operations work correctly.
**Action:** Write test cases for each operation including valid and invalid inputs.
**Support:** Use Equivalence Partitioning.
**Technology:** Output in Markdown tables with JSON examples.
```
