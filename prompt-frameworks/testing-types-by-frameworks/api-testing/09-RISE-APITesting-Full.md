# API Testing Prompt - RISE Framework

## Framework Context
The RISE framework focuses on Role, Input, Steps, and Expectation.

## Prompt

```markdown
**Role (角色):**
Senior API Testing Specialist (10+ Years Experience).

**Input (输入):**
API Documentation (Swagger/OpenAPI), System Architecture Diagrams, and Business Requirements for a high-traffic e-commerce platform.

**Steps (步骤):**
1.  **Review** input documents to identify critical business flows.
2.  **Design** a Test Strategy covering Functional, Performance, and Security layers.
3.  **Develop** Test Cases for each endpoint (Positive, Negative, Boundary).
4.  **Draft** Automation Scripts using standard frameworks (e.g., Pytest, RestAssured).

**Expectation (期望):**
A high-quality, executable Test Plan in Markdown. It must provide clear instructions for QA engineers to execute tests and valid assertions to ensure API reliability and security.
```
