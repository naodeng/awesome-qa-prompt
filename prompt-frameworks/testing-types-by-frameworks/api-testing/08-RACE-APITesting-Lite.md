# API Testing Prompt - RACE Framework (Lite)

## Framework Context
The RACE framework focuses on Role/Background, Action, Context/Result, and Example.

## Prompt

```markdown
**R (Role):** Senior API Tester.
**A (Action):** Create test cases for a User Registration API.
**C (Result):** A Markdown table with Test Scenario, Request Body, and Expected Status.
**E (Example):** Scenario: "Duplicate Email", Expected: "409 Conflict".
```
