# API Testing Prompt - APE Framework (Lite)

## Framework Context
The APE framework focuses on Action, Purpose, and Expectation.

## Prompt

```markdown
**Action:** Design a comprehensive API testing strategy as a Senior API Testing Expert covering Functional, Performance, and Security aspects for [REST/GraphQL] APIs.

**Purpose:** Ensure API quality, stability, and security before deployment.

**Expectation:** Output a Markdown testing plan including:
1. Test Strategy (Layers & Priorities).
2. Detailed Test Cases (Positive, Negative, Boundary).
3. Sample Automation Scripts (Postman/Pytest).
```
