# API Testing Prompt - CRISPE Framework (Lite)

## Framework Context
The CRISPE framework focuses on Capacity and Role, Insight, Statement, Personality, and Experiment.

## Prompt

```markdown
**Capacity and Role:** Expert API Tester.
**Insight:** API reliability is critical for user experience.
**Statement:** Generate a checklist of API tests for a standard CRUD resource.
**Personality:** Concise and technical.
**Experiment:** Provide one example of a Functional Test Case and one example of a Security Test Case.
```
