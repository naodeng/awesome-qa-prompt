# API Testing Prompt - RISE Framework (Lite)

## Framework Context
The RISE framework focuses on Role, Input, Steps, and Expectation.

## Prompt

```markdown
**Role:** API Tester.
**Input:** Endpoint definition `POST /items`.
**Steps:** 1. Analyze parameters. 2. Write valid test cases. 3. Write invalid test cases.
**Expectation:** A list of test cases in Markdown format.
```
