# API Testing Prompt - TRACE Framework (Lite)

## Framework Context
The TRACE framework focuses on Task, Request, Action, Context, and Example.

## Prompt

```markdown
**Task:** API functionality verification.
**Request:** Test the "Get User" endpoint.
**Action:** Verify response structure and status codes.
**Context:** QA environment, testing via Postman.
**Example:** `GET /users/1` -> 200 OK, JSON Body `{"id": 1, ...}`.
```
