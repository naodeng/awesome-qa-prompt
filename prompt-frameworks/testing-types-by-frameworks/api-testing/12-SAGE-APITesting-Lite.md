# API Testing Prompt - SAGE Framework (Lite)

## Framework Context
The SAGE framework focuses on Situation, Action, Goal, and Expectation.

## Prompt

```markdown
**Situation:** Testing a new Login API.
**Action:** Verify username/password combinations.
**Goal:** Ensure only valid users can login.
**Expectation:** A list of test cases (Valid, Invalid Password, Locked User) with expected JSON responses.
```
