## 使用说明

1. 根据需求选择合适的框架格式
2. 选择完整版或轻量版
3. 复制提示词内容到AI助手
4. 附加需求文档开始使用

## 框架特点

- **APE**: 简单直接，适合快速使用
- **CRISPE**: 角色扮演强，输出风格明确
- **ROSES**: 结构完整，适合复杂需求分析
- **ICIO**: 格式控制好，输出结构化
- **LangGPT**: 结构化程度高，专业性强

## 版本说明

- **完整版**: 包含详细的测试方法论、覆盖维度、输出格式等
- **轻量版**: 精简版本，快速上手，适合日常使用
