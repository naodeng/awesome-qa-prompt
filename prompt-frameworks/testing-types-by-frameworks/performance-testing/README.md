## 使用说明

1. **选择框架**：根据你的需求和偏好选择合适的框架
2. **选择版本**：
   - **完整版（Full）**：包含详细的方法论、指标体系和完整的性能测试流程
   - **轻量版（Lite）**：保持简洁但完整的结构，适合快速使用
3. **复制使用**：复制选中框架的内容到AI助手中，然后附加你的系统架构和性能需求文档

## 性能测试核心内容

所有框架都包含以下核心内容：

### 专业角色
- 资深性能测试专家，10年以上性能测试经验
- 精通各种性能测试类型和分析方法

### 测试类型
- 负载测试、压力测试、峰值测试
- 容量测试、稳定性测试、可扩展性测试

### 测试分类
- Web应用性能测试
- API接口性能测试  
- 数据库性能测试
- 系统资源性能测试

### 性能指标
- 用户体验指标（响应时间、页面加载时间）
- 系统吞吐量指标（TPS、QPS、并发用户数）
- 系统资源指标（CPU、内存、磁盘、网络）
- 应用性能指标（错误率、可用性）

### 输出格式
结构化的性能测试方案，包含：
- 测试概述和系统架构分析
- 性能需求分析和测试场景设计
- 执行计划和瓶颈分析框架
- 测试报告模板
