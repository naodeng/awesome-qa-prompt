# ROSES Framework

## Concept

```
Role: Specify the role of ChatGPT.
Objective: State the purpose or goal.
Scenario: Describe the situation.
Solution: Define the expected outcome.
Steps: Ask for the actions needed to achieve the solution.
```

The ROSES framework is a structured method for creating ChatGPT prompts, breaking down interactions into five core components.

- **Role**: Specify the role of ChatGPT
- **Objective**: Describe the goal or task to be achieved
- **Scenario**: Provide relevant background information or context
- **Expected Solution**: Describe the expected solution or outcome
- **Steps**: Ask for specific steps needed to implement the solution

## Example

**Test Case: Web Test Case Writing**

```
Role: As a senior web testing engineer
Objective: Write a complete project test case document
Scenario: The project requires comprehensive test coverage, including positive and negative scenarios, to ensure quality
Solution: Write a well-structured, comprehensive test case in markdown format
Steps: Collect project requirements, perform requirements analysis, write test cases, including positive and negative scenarios
```

**Prompt**: You are a senior web testing engineer writing test cases for a project. The project requires comprehensive test coverage, including positive and negative scenarios, to ensure quality. You need to write a well-structured, comprehensive test case in markdown format. First, collect the project requirements, perform requirements analysis, then write test cases, ensuring to include both positive and negative scenarios.

## Applicable Scenarios

- Tasks requiring clear role and scenario definition
- Complex problem-solving scenarios
- Projects requiring step-by-step execution

## Advantages

- Complete framework covering role, objective, scenario, solution, and steps
- Clear structure, easy to understand and execute
- Suitable for complex tasks requiring detailed guidance
