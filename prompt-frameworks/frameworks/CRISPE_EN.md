# CRISPE Framework

## Concept

```
Capacity and Role: The role (or roles) that should be played
Insight: Provide behind-the-scenes insights, background information, and context.
Statement: Clearly and concisely state the task you want completed
Personality: The style, personality, or manner of the response
Experiment: Request multiple answer examples
```

CRISPE was created by Matt Nigh, providing structure and clarity by breaking down the prompt creation process into clear and distinct steps.

- **Capacity and Role**: Define the role ChatGPT should play
- **Insight**: Provide behind-the-scenes insights, background, and context
- **Statement**: State the task you want ChatGPT to complete
- **Personality**: Define the style, personality, or manner to adopt when responding
- **Experiment**: Request multiple examples

## Example

**Test Case: Web Test Case Writing**

```
Capacity and Role: Play the role of a senior web testing engineer.
Insight: Test cases are key to ensuring project quality, need to cover positive and negative scenarios, ensuring comprehensiveness.
Statement: Write a complete project test case document.
Personality: Maintain professionalism and rigor, clear logic, standardized format.
Experiment: Provide multiple test case examples for different scenarios.
```

**Prompt**: "You are a senior web testing engineer who understands best practices for test case writing. Your task now is to write a complete project test case document. Test cases are key to ensuring project quality, need to cover positive and negative scenarios, ensuring comprehensiveness. During the writing process, maintain professionalism and rigor, clear logic, standardized format, using markdown format. First, collect the project requirements, perform requirements analysis, then write test cases, and provide multiple test case examples for different scenarios."

## Applicable Scenarios

- Tasks requiring specific role-playing
- Scenarios with clear requirements for output style
- Situations requiring multiple example references

## Advantages

- Complete framework structure with broad coverage
- Emphasizes role and personality for more targeted output
- Encourages diverse output through experimental elements
